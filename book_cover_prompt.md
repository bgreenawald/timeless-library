# Book Cover Generation Prompt

## Core Structure

Create a minimalist book cover with three horizontal color blocks:

- **Top block (60%)**: Book title in modern sans-serif
- **Middle block (25%)**: Author name(s)
- **Bottom block (15%)**: "Timeless Library Edition"

Title and bottom blocks use the same color; middle block contrasts.

## Reference & Inspiration

Use the provided example cover as your foundation for:

- Overall aesthetic and modern visual style
- Typography hierarchy and spacing approach
- General composition principles

**Be creative with variations while maintaining visual consistency with the
example.**

## Color Palette

Use colors from this Timeless Library theme palette:

**Primary Palette:**

- Warm cream/off-white: `rgb(250, 243, 232)` - #FAF3E8
- Soft dark gray: `rgb(58, 58, 58)` - #3A3A3A
- Muted gold: `rgb(180, 140, 79)` - #B48C4F
- Darker gold: `rgb(156, 120, 66)` - #9C7842
- Subtle border gray: `rgb(229, 224, 215)` - #E5E0D7

**Color Rules:**

- Use 2-3 colors maximum per cover
- Ensure high contrast between text and background (4.5:1 minimum)
- Avoid pure black/whiteâ€”use the palette's softer tones

## Typography

- Modern sans-serif font family
- Scale appropriately: large title, medium author, small edition
- Maintain clean spacing and readability

## Creative Variations

Choose 1-2 per cover:

- Adjust block proportions (55-65% / 20-30% / 15%)
- Add minimal geometric accents (small shapes, thin divider lines)
- Vary text alignment or typography treatments (italics, small caps, creative
  line breaks)
- Apply subtle textures or borders (5-10% opacity)
- Explore different color combinations from the palette

## Technical Specs

- Aspect ratio: 2:3
- High resolution (1200x1800px minimum)
- Clean, vector-style design
- Maximum 1-2 visual elements beyond the three-block structure

## Quality Standards

- Professional, library-appropriate aesthetic
- Excellent readability at all sizes
- Modern yet timeless
- No visual clutter
